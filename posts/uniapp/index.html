聚名插槽

假如页面的某个子组件内部需要使用多个插槽，直接通过上述方法是不够的，理论上我们需要给每一个插槽去一个名来标记，否则，所有的插槽都会被插入。

  • 子组件写法如下
<template>
	<view>
		<view class="hander">
			<slot name="header"><!--通过插槽在这个组件中预留一个区域--></slot>
		</view>
		<view class="main">
			<slot name='main'><!--通过插槽在这个组件中预留一个区域--></slot>
		</view>
		<view class="footer">
			<slot name="footer">
				
			</slot>
		</view>
	</view>
</template>
  • 父组件写法
<template>
	
<ytw-layout>
	<!-- 可以直接追加插槽名称 -->
	<template v-slot:header>  
		头部内容.....
	</template>
	<!-- 第二种写法 -->
	<template #main>
		主体内容......
	</template>
</ytw-layout>
</template>

这样就能根据插槽的名称，指定插槽的形式在父组件中插入内容

emit声明事件触发

通过emit在子组件声明一个事件，它可以在父组件中声明并触发，从而实现将数据从子组件传递到父组件的功能

  • 子组件
<template>
	<view>
		子组件内容！！！！！！
		<button @click="$emit('add',123)">按钮1</button>
		<!-- 第二种写法 -->
		<button @click="doClick">按钮2</button>
	</view>
</template>
<script setup>
	// 第二种写法声明事件名称
const emit = defineEmits(["del"]);
function doClick(){
	emit("del","删除")
}
</script>

</style>

 # 可以看到子组件的按钮点击时间通过emit声明了一个add事件并且传递了参数123
  • 父组件
<template>
<ytw-child @add="onAdd"></ytw-child>
</template>

<script setup>
const onAdd = function(e){
	console.log(e)
}
</script>

</style>

# 可以看到父组件上讲add事件的回调声明为onAdd，这样，一旦子组件点击后执行顺序如下
子组件触发点击事件=>声明add事件=>父组件上ytw-child标签上的add事件触发=>触发add时间的回调onAdd 执行了打印e的动作，看到了传递的的参数123

6 使用defineExpose暴露子组件的属性及方法

上述的组件中虽然可以通过emit来向外声明函数，但是外部依旧无法直接访问子组件内部的数据，因此vue提供的defineExpose就可以在uniapp中很好的进行导入了。

  • 子组件
<template>
	<view class="layout">
		ytw-demo组件
		{{"子组件count值"+count}}
	</view>
</template>

<script setup>
	import {
		ref
	} from 'vue';
	const count = ref(100)
	const name = ref("余天王")
	const obj = ref({
		age: 111,
		price: 9999
	})
	function countAdd() {
		count.value++
	}
	defineExpose({
		count,
		name,
		obj,
		countAdd
	}) // 向外导出,以便父组件进行导出
</script>

<style lang="scss" scoped>
.layout{
	background-color: green;
	height: 200px;
}
</style>
  • 父组件
<template>
<view class="layout">
	<ytw-demo ref="child1"></ytw-demo>  
	<!-- 子组件通过defineExpose暴露的数据都会通过这个child1暴露出来，只需要在js中定义同名变量，在页面挂载后就会将其自动赋值 -->
	<button @click="child1.countAdd">父组件按钮</button>
</view>
</template>

<script setup>
import { onMounted, ref } from 'vue';
const child1 = ref(null);
console.log("child.value:",child1.value)  // 直接打印因为组件还么有挂载，因此只是null
onMounted(()=>{ // 页面渲染后就会将对应的child暴露给父组件
	// console.log("onMounted打印结果:",child1.value)
	console.log("onMounted打印结果:",child1.value.count)
	console.log("onMounted打印结果:",child1.value.name)
	console.log("onMounted打印结果:",child1.value.obj)
	console.log("onMounted打印结果:",child1.value.countAdd)
})
</script>
